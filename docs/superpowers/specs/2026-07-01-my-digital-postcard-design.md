# my-digital-postcard

A small browser app that turns a photo into a downloadable postage stamp. You either
upload a photo or take one with your webcam, pick a film look, and the app grades it
like real film and drops it into a stamp frame you can save as a PNG.

The whole point is that the film looks feel like a real camera, not a slider. You tap
one look and it just works. No color grading knobs.

## Goals

- Real film looks that read as "shot on film", built from actual .cube LUTs plus
  camera artifacts (grain, halation, bloom, vignette), not CSS filters.
- Live camera capture that feels like taking a photo: freeze the frame, then grade it.
- Photo upload as the other way in.
- Output is a stamp: perforated edges, paper texture, a title, and a number. Full color
  photo inside. Downloadable PNG.

## Non goals (for now)

- No accounts, no backend, no saving to a server. Everything runs in the browser.
- No manual grading sliders. Looks are one tap.
- Not a mobile app. It runs in a desktop browser.

## Stack

- Plain HTML, CSS, and vanilla JS modules. No framework, no bundler.
- WebGL2 for the render pipeline (real 3D texture LUT sampling, multi pass artifacts).
- Runs from a tiny static server because .cube files are fetched (file:// blocks fetch).

## How it is built

The app is split into small pieces that each do one job.

- `js/camera.js` opens the webcam with getUserMedia and grabs a still frame on capture.
- `js/upload.js` handles file pick and drag and drop, decodes to an image.
- `js/gl/lut.js` parses a .cube file into a 3D LUT texture.
- `js/gl/shaders.js` holds the GLSL sources for each pass.
- `js/gl/renderer.js` sets up WebGL2, runs the pass pipeline with framebuffer ping pong,
  and returns the graded image as a texture or canvas.
- `js/presets.js` defines each film look: which LUT to load and the artifact settings.
- `js/stamp.js` composes the final stamp on a 2D canvas: photo, perforations, paper,
  title, number, date stamp, then exports a PNG.
- `js/main.js` wires the UI: pick a source, pick a look, capture or grade, download.

## The render pipeline (the important part)

Input image goes through these passes in order. Heavy passes run on the frozen still, not
on live video, so it stays fast and the export is sharp.

1. Base draw of the source image to a framebuffer.
2. LUT color pass. Sample the 3D LUT with trilinear interpolation. This is the film
   color science. Applied at preset strength, around 0.7 to 0.9, so it never looks
   overcooked.
3. Highlight extract plus blur, reused for two things:
   - Halation: tint the blurred highlights warm and screen them back on. This is the
     red glow around bright edges that Kodak film has.
   - Bloom: soft glow spilling from the highlights.
4. Grain. Animated, luminance modulated noise so it is stronger in the mids and quiet in
   the highlights. This is the number one tell of real film.
5. Vignette. Gentle edge darkening.
6. Optional date stamp and light leaks. Both off by default. Date stamp is an orange
   seven segment style number in the corner.

## Film looks (ship these three first)

Each look is a real .cube LUT plus artifact settings. The LUTs are generated by a small
script so the color science is authored, repeatable, and easy to tweak.

- Kodak warm. Warm skin, muted olive greens, a pale mint cyan in the highlights, soft
  lifted contrast, a little halation. Based on the Kodak film inspo.
- Overcast teal. Cold and desaturated, misty cyan green, low contrast, soft blacks. The
  quiet muted street look. Reds still pop because everything else is muted.
- Golden hour. Deep warm directional light, richer contrast, warm highlights against
  cooler shadows.

## The stamp output

The stamp is the identity of the app, so it gets real craft, not just a border.

- Perforated edge made from real punched half circles around the frame.
- Off white paper with a faint texture.
- A title you can type, a big number, and a small caption line with the date.
- Full color graded photo fills the stamp window.
- Optional halftone print treatment as a toggle for a printed ink feel. Off by default so
  the film color stays intact.
- Export a high resolution PNG.

## Running it

A one line static server, for example `npx serve` or `python3 -m http.server`. Open the
page, allow the camera if you want live capture, or drop in a photo.

## Scope for the first build

Ship the full pipeline with the three looks, upload plus camera capture, grain and
halation and bloom and vignette, and the stamp export. Extra looks and extra artifacts
come later.
